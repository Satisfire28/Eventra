# Eventra 🎟️

Full-stack MERN event booking platform

## Features

- JWT Authentication
- Email OTP verification
- Admin dashboard
- Event booking system
- Role-based access
- MongoDB Atlas integration
- Seed data generator

## Tech Stack

Frontend:
React + Tailwind + Axios

Backend:
Node.js + Express + MongoDB

Deployment:
Render

## Setup

Clone repo:

npm install

Run:

npm run dev
